## Tentang
Proyek ini menunjukkan bahwa kita bisa menyisipkan "animasi tanpa teks" (seperti sprite karakter game) langsung di dalam file `README.md`. 
Animasi slime di atas dibuat menggunakan gambar berformat SVG (Scalable Vector Graphics) yang memiliki instruksi pergerakan secara internal.

Karena dibuat tanpa script eksternal, format ini sangat cocok dan aman untuk ditampilkan di GitHub atau platform Markdown lainnya.
